Job Description

Role: Associate DevOps Engineer – NexusOne Location: Pune Category: Platform Engineering & DevOps Reports To: DevOps Engineering Manager Role Summary / Purpose As an Associate DevOps Engineer at NexusOne, you’ll help design, automate, and optimize the infrastructure that powers our global data and analytics platform. This role is ideal for recent graduates who are passionate about cloud infrastructure, CI/CD automation, and modern DevOps practices. You’ll work alongside experienced engineers to support deployment pipelines, observability tools, and infrastructure-as-code environments — ensuring NexusOne delivers secure, scalable, and reliable systems to customers worldwide. Core Responsibilities ● Assist in building and maintaining CI/CD pipelines using tools like GitLab CI, Jenkins, or GitHub Actions. ● Support the provisioning and management of cloud infrastructure (AWS, Azure, or GCP) using Terraform or similar IaC tools. ● Help automate environment setup, application deployments, and monitoring processes. ● Contribute to containerized application environments using Docker and Kubernetes. ● Participate in monitoring and incident response efforts by learning and using tools such as Prometheus, Grafana, and ELK Stack. ● Collaborate with software, data, and platform teams to streamline development-to-production workflows. ● Document automation scripts, deployment processes, and environment configurations. ● Learn and apply DevOps best practices for security, scalability, and reliability. Required Skills & Qualifications ● Education: Bachelor’s degree in Computer Science, Information Technology, Engineering, or a related technical field. ● Languages & Scripting: Proficiency in Python, Bash, or similar scripting languages. ● Version Control: Basic experience with Git and collaborative development workflows. ● Foundational Knowledge: Understanding of cloud platforms, CI/CD concepts, and Linux-based systems. ● Problem-Solving: Strong troubleshooting and analytical skills. ● Collaboration: Good communication and eagerness to work in an Agile, team-oriented environment. Nice-to-Have Skills ● Familiarity with Docker and Kubernetes concepts. ● Exposure to Terraform, Ansible, or CloudFormation for infrastructure automation. ● Understanding of networking, system security, or cloud IAM fundamentals. ● Experience with log management and monitoring tools (Grafana, Prometheus, Datadog, etc.). ● Internship or academic project experience in DevOps, cloud, or system automation. Success Criteria ● Demonstrates growing proficiency in CI/CD and infrastructure automation. ● Contributes to stable, repeatable, and secure deployment processes. ● Shows initiative in learning new DevOps tools and cloud technologies. ● Works collaboratively with cross-functional teams to improve delivery efficiency. ● Produces high-quality documentation and configuration transparency.
