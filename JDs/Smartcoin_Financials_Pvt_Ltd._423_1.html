Roles and Responsibilities 1. Core Development Build and enhance Olvy’s backend platform that processes terabytes of data to create a personalized financial experience for users. Write clean, efficient, and maintainable code in Java, applying strong knowledge of data structures and object-oriented design. Implement RESTful APIs following SOLID principles and modern software engineering practices. 2. System Architecture & Design Collaborate with the tech-focused founding team and IIT alumni engineers to design scalable, secure, and high-performance systems. Work within the MVC architecture to build modular and reusable code components. Participate in system design discussions and contribute ideas to improve architecture and scalability. 3. Application Excellence Ensure the platform’s security, scalability, and reliability through best coding and design practices. Debug and optimize existing modules to improve application performance. Debug and optimize queries to ensure high database performance and minimal latency. 5. Ownership & Collaboration Take complete ownership of assigned features and deliver high-quality outcomes. Collaborate closely with cross-functional teams to align technical execution with business objectives. Continuously learn, explore, and apply new technologies to enhance the product and user experience.
