PLACEMENT Embedded System Engineer (Communication) 1400000 INR

Job Description

Description Embedded Systems Engineer with expertise in both embedded software development and hardware integration. Responsible for designing, programming, integrating, and testing embedded systems, with a focus on communication interfaces and device-level control. Understanding of firmware, device drivers, and debugging will be preferred. Responsibilities Develop and maintain embedded firmware using Embedded C/C++ for microcontrollers. Implement and debug device drivers for peripherals (UART, SPI, I2C, GPIO, ADC, etc.). Integrate hardware components (sensors, transceivers, communication modules) and ensure reliable system-level operation. Perform interface testing, and real-time debugging using oscilloscopes, logic analyzers, and JTAG tools. Optimize embedded software for performance, memory, and power efficiency. Interface with cross-functional teams (RF, PCB, and mechanical) for system integration. Prepare and maintain technical documentation, including schematics review notes, software architecture, and test logs. Automate testing and validation processes using Python/Bash scripts.
