Job Description

Job Description – Assistant Manager (Computer Science) Specializations: Cyber Security / Artificial Intelligence CTC: ₹10,00,000 per annum Leaves: 12 per year Department: IT Security / AI & Automation Reporting To: Respective Reporting Manager Role Overview The Assistant Manager (Computer Science) will support the organization’s technology operations in one of the following specializations: 1. Cyber Security Responsible for monitoring, analyzing, and strengthening the organization’s cybersecurity posture, ensuring data protection, incident response, and statutory compliance. 2. Artificial Intelligence (AI & Data Engineering) Responsible for developing AI/ML models, supporting digital automation initiatives, and analyzing organizational data to enable data-driven decision making. Key Responsibilities A. Cyber Security Responsibilities Monitor and analyze security events, logs, and threat intelligence. Conduct Vulnerability Assessment & Penetration Testing (VAPT). Manage security tools such as antivirus, firewalls, SIEM, IDS/IPS. Respond to cybersecurity incidents and support forensic investigations. Ensure security compliance across network, servers, and endpoints. Develop cybersecurity awareness programs for internal teams. Assist in internal/external audits, including ISO/IEC 27001 compliance. B. Artificial Intelligence Responsibilities Build and train Machine Learning and Deep Learning models. Perform data preprocessing, feature engineering & model tuning. Support automation projects using AI/ML technologies. Develop dashboards, predictive analytics reports, and visualizations. Deploy AI models using Python, SQL, and cloud platforms. Conduct in-depth data analysis and provide business insights. Work with cross-functional teams on AI-driven technology initiatives. Required Skills & Tools Cyber Security Skillset VAPT tools: Nmap, Burp Suite, Nessus, Metasploit. Strong understanding of networking, Linux, cloud security. Knowledge of encryption, authentication, and incident response. Artificial Intelligence Skillset Proficiency in Python, ML algorithms, Deep Learning, NLP. Tools: TensorFlow, PyTorch, Scikit-learn. Strong understanding of data structures, databases, cloud (AWS/Azure/GCP). Eligibility Criteria B.Tech/B.E. in Computer Science / IT / AI / Data Science. For Cyber Security: Certification preferred – CEH, CompTIA Security+, OSCP. For AI: Internship or academic project experience in ML/AI preferred. Excellent analytical ability and problem-solving skills.
