Data Engineer Technologies / Skills: Advanced SQL, Python and associated libraries like Pandas, NumPy etc., PySpark , Shell scripting, Data Modeling, Big data, Hadoop, Hive, ETL pipelines. Responsibilities: • Communicate with users, other technical teams, and senior management to collect requirements, describe data modeling decisions and develop data engineering strategy • Work with business owners to define key business requirements and convert to user stories with required technical specifications • Convey results and business impacts of insight initiatives to key stakeholders to collaboratively solve business problems • Collaborate closely with the overall Enterprise Data & Analytics Architect, Engineering practice leads to ensure adherence with the best practices and design principles • Assure quality, security and compliance requirements are met for supported area • Design and create fault-tolerance data pipelines running on cluster • Persuade client business and IT teams to adopt cutting edge technology • Design data engineering solutions end to end. Come up with scalable and modular solutions Required Qualification: ➢ Designing and developing Data Pipelines for Data Ingestion or Transformation using Python (PySpark)/Spark SQL in AWS cloud ➢ Design and development of data pipelines and processing of data at scale ➢ Writing and optimizing efficient SQL queries with Python and Hive handling Large Data Sets in Big-Data Environments ➢ Debugging, tunning and optimizing PySpark data pipelines ➢ PySpark data frames, joins, caching, memory management, partitioning, parallelism etc. ➢ Spark UI, Event Timelines, DAG, Spark config parameters, in order to tune the long running data pipelines ➢ Agile implementations ➢ Build data pipelines in streaming and batch mode ➢ Git and CI/CD pipelines to deploy cloud applications ➢ Good knowledge of designing Hive tables with partitioning for performance Desired Qualification: ❖ Understanding of data modeling ❖ Hands on creating workflows on any Scheduling Tool like Autosys, CA Workload Automation ❖ Proficiency in using SDKs for interacting with native AWS services ❖ Strong understanding of concepts of ETL, ELT and data modeling Data Scientist Responsibilities: • Develop and enhance forecasting models using time series forecasting, statistical modeling, and machine learning techniques • Analyse historical and real-time data to identify trends and improve forecasting accuracy • Investigate discrepancies between forecasted and actual data, providing actionable insights • Automate data pipelines and forecasting workflows to streamline processes across departments • Monitor day-over-day forecast variations and communicate key insights to stakeholders • Work closely with internal teams and external vendors to refine forecasting methodologies. • Perform scenario analysis to assess seasonal patterns, anomalies, and market trends. Continuously optimize forecasting models, leveraging techniques like ARIMA, Prophet, LSTMs, and regression-based models Required Qualifications: ✓ Knowledge in data science ✓ Strong expertise in time series analysis, forecasting algorithms, and statistical modeling. ✓ Proficiency in Python, with experience using libraries such as pandas, NumPy, scikit-learn, stats models, and TensorFlow/PyTorch. ✓ SQL and handling large datasets. Hands-on with forecasting models like ARIMA, SARIMA, Prophet, LSTMs, XGBoost, and random forests. ✓ Familiarity with feature engineering, anomaly detection, and seasonality analysis. Strong analytical and problem-solving skills with a data-driven mindset. ✓ Excellent communication skills, with the ability to translate technical findings into business insights.
