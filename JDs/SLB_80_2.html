Full Stack Software Engineer Job Description: We are looking for a passionate Software Engineer to design & develop software solutions. Software Engineer responsibilities include gathering user requirements, defining system functionality and writing code in various languages, like Java, GoLang , Angular programming languages Our ideal candidates are familiar with the software development life cycle (Agile) from preliminary system analysis to tests and deployment. Ultimately, the role of the Software Engineer is to build high-quality, innovative and fully performing software that complies with coding standards and technical design. Responsibilities: • Execute full software development life cycle (Agile) • Write well-designed, testable code • Produce specifications and determine operational feasibility • Integrate software components into a fully functional software system • Develop software verification plans and quality assurance procedures • Document and maintain software functionality • Troubleshoot, debug and upgrade existing systems • Comply with project plans and industry standards Requirements: • Ability to develop software in Java, GoLang, Angular programming languages • Excellent knowledge of relational databases, SQL and ORM technologies (JPA2, Hibernate) • Experience with test-driven development • Proficiency in software engineering tools • Ability to document requirements and specifications SLB is an equal employment opportunity employer.
