Job Description

Summary As an Associate Data Science Engineer, you will transform raw data into actionable insights, integrate machine learning models along with utilizing a combination of data engineering, full-stack development, and continuous learning for comprehensive and impactful solutions. Responsibilities  Data Collection and Ingestion: Write pipelines by using various data engineering guidelines to enable data flow from disparate sources and making is available at a central location for further analytical use-cases using Cloud / Hybrid Eco-systems.  Data Exploration and Insight Generation: Use large and complex datasets for churning out meaningful insights and make it consumable in form of visualisation. Usage of tech. like – Python, Pandas, Spark, Data Factory, Databricks, DWH will be the daily job.  Programming and Software Development: Utilize Python and Full-Stack (HTML, CSS, JS) skills to perform data API development, Data Application development which enhances data consumptions. Integration of models in Data Pipelines, Data Apps will be needed for making a business focused analytical solutions.  Infrastructure and Architecture: Good understanding of Computer Hardware is must along with Cloud understanding, collaboration with the Infra team is needed to implement scalable data solutions.  Data Visualisation: Creation of interactive visualisation is needed using D3js, Chart.js, Vega.js etc.  Data Quality: Monitor data quality and help implement data governance practices to ensure data integrity.  Continuous Learning: Stay up-to-date with industry trends and emerging technologies related to data analysis. Job Title: Associate Data Science Engineer   Documentation: Maintain detailed documentation of all the technical things. Qualifications   Bachelor's (4 years)/ Masters (3+2 years) degree from a premier institution in a relevant field (e.g., Computer Science, Information Technology, MCA or other equivalent streams).   Strong analytical and problem-solving skills.   Proficiency in tools such as Excel, Python, SQL, HTML, CSS, JS, Data Factory, Databricks will be an added advantage.   Basic understanding of DevOps concepts, AI/ML will be helpful.   Excellent communication and presentation skills.   Ability to work both independently and collaboratively within a team.   Eagerness to learn and adapt to new technologies and tools.   A passion for data-driven decision-making.
