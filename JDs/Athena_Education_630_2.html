Job Description

We are looking for a highly skilled AI/ML Engineer Mentor who can architect, develop, and deploy intelligent systems across computer vision, NLP, predictive analytics, and GenAI applications. This role requires a hands-on engineer with strong research awareness, deep technical expertise, and the ability to mentor high-potential high-school scholars and interns in applied artificial intelligence. You will be responsible for guiding AI/ML projects end-to-end—from ideation and experimentation to deployment and evaluation—while driving innovation within the Athena AI Lab. Key Responsibilities 1. AI/ML System Development Design and implement machine learning pipelines using Python, Scikit-Learn, TensorFlow, PyTorch, Keras, and JAX. Develop models for: Computer Vision (classification, detection, segmentation) NLP (transformers, summarization, semantic search, text classification) Time-series forecasting, anomaly detection & applied analytics Generative AI (LLMs, diffusion models, prompt engineering) Experiment with state-of-the-art architectures and integrate modern techniques from top university research (Ivy League labs, MIT CSAIL, Stanford HAI, CMU ML Lab). 2. LLMs & Generative AI Build and fine-tune large language models (open-source LLMs such as Llama, Mistral, Gemma). Develop generative pipelines for: Text generation Code assistants Vision-language tasks Embeddings, retrieval, and hybrid search Implement RAG systems, vector databases, and multi-agent workflows. 3. MLOps, Deployment & Productization Deploy models to scalable platforms: FastAPI, Flask, Docker, Kubernetes, AWS/GCP/Azure ML Set up end-to-end ML lifecycle: CI/CD, model registries, experiment tracking (MLflow, Weights & Biases) Optimize models for real-time or resource-restricted production environments. 4. Data Engineering & Pipeline Management Build robust data pipelines for ingestion, preprocessing, augmentation, validation, and monitoring. Work with structured, semi-structured, and unstructured datasets. Implement data quality checks, drift detection, and labeling workflows. 5. Mentoring & Scholar Guidance Mentor high-school scholars in AI/ML fundamentals and guide them in building impactful AI-based projects. Teach students how to: Frame AI problems Work with datasets Train and evaluate ML models Build end-to-end applications Conduct workshops on ML, Python, RAG, CV/NLP, and AI product thinking. 6. Research, Innovation & Documentation Stay updated with latest advancements from elite AI research labs. Translate cutting-edge research into practical learning modules and project ideas. Maintain high-quality documentation, experiment logs, model cards, and reproducibility assets. Technical Requirements (Latest & Industry/Research Aligned) Must-Have Technical Skills Strong programming skills in Python; proficiency in Java is a plus. Deep understanding of: Machine Learning & Deep Learning algorithms Neural networks, optimizers, loss functions Transformer architectures & attention mechanisms Hands-on experience with: TensorFlow / PyTorch Scikit-learn OpenCV Hugging Face transformers Solid foundation in Data Structures & Algorithms. Experience with: REST APIs Web/Mobile app integration Backend deployment (FastAPI, Flask) Strong communication skills to interact with high-school scholars.
