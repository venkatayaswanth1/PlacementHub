Job Description

DAERO is looking for a Swift / iOS Engineer to build and scale DAERO’s core mobile experience - from UI design and system architecture to backend integration and performance optimization. You’ll own the end-to-end iOS stack, ensuring our app is fast, reliable, and reactive across devices. Responsibilities Develop, optimize, and maintain iOS applications using Swift and SwiftUI. Architect clean, modular systems that integrate seamlessly with backend APIs. Lead A/B testing, unit testing, and ensure smooth performance on lower-end devices. Collaborate on system design decisions that influence scalability and user experience. Implement CI/CD pipelines, analytics hooks, and reactive UI patterns for production-grade apps. Requirements Has built multiple iOS apps in the past. Strong grasp of system design, asynchronous patterns, and API architecture. Hands-on experience with Combine, Core Data, Swift Concurrency, and UI performance optimization. Knowledge of CI/CD (Fastlane, GitHub Actions) and App Store deployment workflows. Bonus Experience with Reactive architectures, GraphQL, or real-time streaming APIs. Exposure to AI-driven UI components or speech/audio interfaces. Interest in product design, usability testing, and data-informed iteration. Familiarity with Kotlin Multiplatform Mobile (KMM) or cross-platform integration is a plus.
