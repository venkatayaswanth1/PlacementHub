Job Description

Key Responsibilities: • Data-Driven Consulting: Work on consulting projects for global clients, focusing on innovation and solving business problems using data. • Learning and Development: Quickly adapt and learn new skills on the job, with a proactive and passionate approach to understanding new technologies and methodologies. • Development and Deployment of AI Solutions: Assist in building, training, and optimizing machine learning, deep learning, and generative AI models to address various business needs. This includes conducting exploratory data analysis, validating and testing models for accuracy and reliability, and participating in the deployment and continuous improvement of AI solutions. • Team Collaboration: Work closely with interdisciplinary teams, including data scientists, data engineers, product developers, and client managers, to achieve project goals. • Communicating Technical Insights: Translate complex technical concepts into simple, actionable insights for non-technical stakeholders, ensuring clarity and understanding. • SQL & Database Management: Write and optimize SQL queries, perform database tuning, testing, and troubleshoot issues. Ensure data platform security, backup, recovery, and performance optimization. Skill Requirements: Essential • Programming and Algorithms: Practical experience with at least one programming language (e.g., Python, Java, C++). Basic knowledge of algorithms is required. • Problem-Solving Skills: Strong analytical and problem-solving skills with a willingness to think creatively and innovate. • Communication and Collaboration: Excellent communication skills and an ability to work effectively in a multidisciplinary team environment. • Database Knowledge: Familiarity with relational databases (RDBMS) (eg. mySQL) or non-relational (No-SQL) databases. • Linux Fundamentals: Basic understanding of Linux concepts and tools, including virtual environments (e.g., virtualenv), containerization (e.g., Docker), and version control using Git for code management. • Data and Machine Learning Basics: Understanding of basic statistical concepts, data structures, and introductory knowledge in areas like Natural Language Processing (NLP), computer vision, or speech recognition. Preferred: • Machine Learning & Data Science Libraries: Exposure to ML/DL libraries (e.g., TensorFlow, PyTorch) and basic concepts in Data Science, Machine Learning, and Deep Learning. • Generative AI & Open-Source Tools: Experience with open-source tools such as Hugging Face, LangChain, and an understanding of prompt engineering for generative AI is a plus. • Cloud Computing: Familiarity with cloud platforms like AWS, Azure, or GCP. • Projects & Certifications: Prior projects, internships, or certifications related to data processing, analytics, cloud computing, or relevant courses in data, cloud, or analytics.
