Job Description

Role: Associate Product Engineer – NexusOne Location: Global Category: Product Engineering & Platform Development Reports To: Product Engineering Manager Role Summary / Purpose As an Associate Product Engineer at NexusOne, you’ll contribute to the design, development, and delivery of innovative features that power our open, composable data and AI platform. This is an exciting opportunity for recent graduates who are passionate about building scalable software products, solving complex technical problems, and collaborating in an agile, cross-functional environment. You’ll work closely with product managers, designers, and other engineers to turn ideas into impactful, user-focused solutions. Core Responsibilities ● Develop, test, and maintain front-end and/or back-end components of the NexusOne platform. ● Collaborate with senior engineers to design scalable APIs, data services, and microservices. ● Participate in code reviews, sprint planning, and agile development ceremonies. ● Contribute to user interface development using modern frameworks (e.g., React, TypeScript). ● Implement server-side logic and integrations using technologies such as Python, Node.js, or Java. ● Write automated tests and follow best practices for clean, maintainable code. ● Support continuous integration and deployment through modern DevOps tools. ● Document features, APIs, and technical decisions to ensure clarity and collaboration. Required Skills & Qualifications ● Education: Bachelor’s degree in Computer Science, Software Engineering, or a related technical discipline. ● Languages: Proficiency in one or more programming languages such as Python, JavaScript/TypeScript, or Java. ● Software Fundamentals: Strong grasp of algorithms, data structures, and object-oriented programming. ● Development Tools: Experience with Git, VS Code, and modern build tools. ● Collaboration: Strong communication skills and ability to work in an agile team setting. ● Mindset: Curiosity, adaptability, and a desire to learn from experienced engineers. Nice-to-Have Skills ● Experience building web applications with frameworks like React, Next.js, or Angular. ● Exposure to backend frameworks such as FastAPI, Spring Boot, or Express.js. ● Familiarity with RESTful API design and cloud-based development (AWS, Azure, or GCP). ● Understanding of relational and NoSQL databases (e.g., PostgreSQL, MongoDB). ● Internship, co-op, or project experience developing software products or APIs. Success Criteria ● Delivers reliable, maintainable, and well-documented code. ● Demonstrates continuous learning and growth in both front-end and back-end development. ● Collaborates effectively with designers, product managers, and other engineers. ● Shows initiative in improving code quality, performance, and user experience. ● Contributes to features that enhance the value and usability of the NexusOne platform.
